存储器组织

8086/8088是16位微处理器，但有二十根地址线，能寻1MB的内存空间，离不开分段结构。

分段结构

1MB(即 或 )的内存空间，划分为若干个大小为64KB(即 或 )的逻辑段空间。"逻辑"的含义是指，每个段可以在整个存储空间浮动，允许部分重叠、完全重叠、连续排列。

这意味着同一个固定的物理地址，有可能同时包含在在不同的段中，有各自不同的偏移地址(见下文)。

段地址、段内地址(偏移地址)和物理地址

从上文可以看出，上面阐述了分段的机制，那么段内同样存在一个相对段起始位置的偏移量，我们叫做段内地址(偏移地址)。段地址是指逻辑段的物理地址前四位。段地址配上段内地址，我们叫做逻辑地址(XXXX:XXXX)

物理地址即在1MB空间中的绝对位置。一般情况下，操作数的段地址由段寄存器提供，而段内地址由指令的寻址方式提供。运算过程如下：

例如段地址为2000H，偏移地址是FFFFH，则物理地址为2FFFFH。

段寄存器有CS、DS、ES。指针寄存器有IP、BP、SP。

对于超出一个字节的数据进行存储，8086规定了低字节存储于低地址，高字节存储于高地址。

例如：将数据4567H存入存储器，我们要使用DS寄存器，已知DS的值为2000H，偏移地址是1000H。则有数据67H的物理地址是21000H，数据45H的物理地址是210001H.

工作模式

8086/8088的CPU有两种工作模式。

最小模式是指微机系统中只有8086微处理器，所有的总线控制信号直接由CPU发出，系统中的总线控制逻辑电路被减到最少。

最大模式是相对而言的，一般微机系统中至少两块处理器，一块8086作为主处理器，一块协处理器，还有各种片外电路。协处理器一般有两种：(1).数值协处理器8087，实现多类型高精度的数值计算。(2).输入输出协处理器8089，有一套专门用于输入输出操作的指令系统，类似于带两个DMA通道的处理器。

引脚信号和功能

8086/8088都是双列直插式集成电路芯片，共40个引脚，其中32个在两种工作模式下名称和功能一样，还有8个则有不同的名称与用处。

：8086中是地址/数据分时复用的信号线，8088中低八位不变，高八位则仅为地址线。分时复用的原理是在CPU内部采用分时多路开关，利用定时信号来区分当前引脚上的信号代表地址还是数据。

在8086/8088CPU执行指令的过程中，一个总线周期的 时刻通过其送出地址信号， 时刻才输入输出数据，若在CPU外部配置一个地址锁存器，用锁存器的输出去选通存储器或者外设端口，那分时复用将更加灵活。

：地址/状态复用、三态输出的引线。

在一个总线周期内的 时刻， 送出的是地址的高四位。在 时刻， 送出状态位信号。和组合用于指示CPU正在使用的段寄存器(见下表)。 输出状态寄存器中中断允许标志位。始终为低电平，代表CPU此时与外部总线相连。

所代表的段寄存器
0 0 附加段寄存器ES
0 1 堆栈段寄存器SS
1 0 代码段寄存器CS
1 1 数据段寄存器DS

当CPU用于数据输入输出时，这四个引脚均为低电平，不作使用。在复位或DMA操作中，将处于高阻态状态(浮空或三态)

INTR：可屏蔽的中断请求输入信号。高电平输入将触发中断。若标志位IF置0时，将屏蔽该引脚的中断请求。(CPU在每条指令执行的最后一个状态采样该信号)

NMI：非屏蔽的中断请求输入信号。边沿触发，正跳变有效。(当输入由低变高时，CPU将在现行指令执行结束后，执行中断类型码为2的不可屏蔽的中断处理程序)

(Buffer High Enable)：高8位数据允许/状态复用引脚。
